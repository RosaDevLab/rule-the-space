# 🚀 Rule The Space

A Python game development series where we build a space-themed game step by step using **Pygame**.

This repository contains the full source code for each episode of the series.

---

## 🎮 About the Project

In this series, we create a 2D space game from scratch using Python and Pygame.

We cover topics like:
- Game window setup
- Player movement
- Background and assets
- Collision detection
- Obstacles and gameplay mechanics

## 🧠 Tech Stack
